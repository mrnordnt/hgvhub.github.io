Getting Started with GitHub & Python

A Beginner’s Guide to Sharing Your Projects

Chapter 1: What is GitHub?

GitHub is like a magical online library—or an online treasure chest—where people store, share, and work on projects together. It’s mostly used for coding, but you can use it for anything—like writing stories, making games, or even this e-book! Think of it as a place to save your work and show it to friends.

Why Use GitHub?

  • Save your projects safely online.
  • Work with others, like a group project.
  • Show off what you’ve made!
Tip: GitHub is free for public projects, so you can start creating right away!

Chapter 2: Setting Up GitHub

Let’s get started!

Make an Account

  1. Go to github.com.
  2. Click “Sign Up” and use your email (ask an adult if needed).
  3. Pick a cool username, like “SuperCoder123” or “PythonWizard123”.

Create a Repository

  1. A “repository” (or repo) is like a folder for your project.
  2. Click the “+” button on GitHub, then “New Repository”.
  3. Name it something fun, like “MyFirstProject” or “MyPythonHub”.
  4. Check “Public” so others can see it, and click “Create Repository”.

Example: Your First Repo

Imagine you’re writing a story. Your repo could be called “SpaceAdventure”. You’ll add files to it soon!

Tip: Keep your repo name short and simple—it’s easier to share!

Chapter 3: Adding Files to GitHub

Now, let’s put something in your repo!

Example 1: Add a Text File

  1. On your repo page, click “Create new file”.
  2. Name it hello.txt.
  3. Type this:
    4. Hi! I’m learning GitHub!
  4. Scroll down, write a message like “Added my first file”, and click “Commit new file”.

What happened? GitHub saved your file! You can edit it anytime by clicking it and choosing “Edit”.

Example 2: A Simple Python Greeting

1. Click “Create new file”.
2. Name it hello.py.
3. Type this:

print("Hello, GitHub! I’m learning Python!")

4. Commit it with a message like “Added hello.py”.

Test it: Copy this code into a Python editor (like IDLE) and run it. What do you see?

Tip: Use clear commit messages (e.g., “Added a greeting”) so others know what you changed!

Chapter 4: Working Together

GitHub lets friends help with your project—it’s awesome for teamwork!

Example 3: Invite a Friend

  1. Go to your repo’s “Settings”.
  2. Click “Collaborators” (you might need a password).
  3. Type your friend’s GitHub username and invite them.
  4. Now they can add files too!

Try this: Ask your friend to add a file called ideas.txt with their favorite game idea.

Example 4: Collaborate on a Python Quiz

1. Create a file quiz.py with:

answer = input("What’s 2 + 2? ")
if answer == "4":
    print("Correct! You’re a math star!")
else:
    print("Oops! Try again next time.")

2. Invite a friend to edit it and add another question!
3. Test it in Python.

Test it: Run this. Type “4” or something else—what happens?

Tip: Check your friend’s changes in the “Commits” tab to see what they added!

Chapter 5: Showing Off with GitHub Pages

Turn your repo into a website—like this e-book!

Example 5: Make a Simple Site

  1. In your repo, create a file named index.html.
  2. Type this:
    3. Welcome to My Page!
    

    This is my GitHub site!
  3. Commit it.
  4. Go to “Settings” > “Pages”, set “Source” to “main” branch, and save.
  5. Visit username.github.io/MyFirstProject after a minute.

Cool, right? You’ve got a mini-website!

Tip: Add some color to your HTML with <p style="color: blue;"> for fun!

Chapter 7: Python + GitHub Project – Number Guessing Game

Let’s make a game and share it!

Example 6: Number Guessing Game

1. Create game.py with:

secret = 6
guess = int(input("Guess a number between 1 and 10: "))
if guess == secret:
    print("You win! Amazing guess!")
else:
    print("Nope! The secret was", secret)

2. Commit it.
3. Ask a friend to change the secret number.

Test it: Run this in Python. Guess a number—what’s the result?

Chapter 8: Exploring More with Python

Python can do cool stuff! Here’s a loop example:

Example 7: Countdown

Add countdown.py to your repo:

for i in range(5, 0, -1):
    print(i)
print("Blast off!")

Test it: Run this—what does it count down to?

Tip: Change range(5, 0, -1) to range(10, 0, -1) for a longer countdown!

How to Host This on GitHub

  1. Create a Repository: Go to GitHub, make a new repo (e.g., GitHubPythonGuide).
  2. Add the E-book: Save this as index.html.
  3. Enable GitHub Pages: Go to “Settings” > “Pages”, set “Source” to “main”, and save.
  4. Share: Visit username.github.io/GitHubPythonGuide and send the link to friends!
Tip: Add a README.md file to your repo with a short description—it’s like a welcome note!
