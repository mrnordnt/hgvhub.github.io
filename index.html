Getting Started with GitHub & Python

A Beginner’s Guide to Sharing Projects and Coding with Python

Chapter 1: What is GitHub?

GitHub is like an online treasure chest where you can store, share, and work on projects with friends. It’s famous for coding, but you can use it for anything—like stories, games, or even this e-book! In this guide, we’ll also mix in some Python coding fun.

Chapter 2: Setting Up GitHub

Let’s get you started!

  1. Go to github.com and click “Sign Up”.
  2. Use your email and pick a username (e.g., “PythonWizard123”). Ask an adult if needed!
  3. Create a repository (repo): Click “+” > “New Repository”, name it (e.g., “MyPythonHub”), make it “Public”, and hit “Create Repository”.
Tip: A repo is like a folder for your project!

Chapter 3: Adding Files to GitHub

Let’s add a Python file to your repo!

Example 1: A Simple Python Greeting

1. In your repo, click “Create new file”.
2. Name it hello.py.
3. Type this:

print("Hello, GitHub! I’m learning Python!")

4. Scroll down, write “Added hello.py” in the commit message, and click “Commit new file”.

Test it: Copy this code into a Python editor (like IDLE) and run it. What do you see?

Chapter 4: Working Together

GitHub is awesome for teamwork!

Example 2: Collaborate on a Python Quiz

1. Create a file quiz.py in your repo with:

answer = input("What’s 2 + 2? ")
if answer == "4":
    print("Correct! You’re a math star!")
else:
    print("Oops! Try again next time.")

2. Invite a friend: Go to “Settings” > “Collaborators”, add their GitHub username.
3. Ask them to edit quiz.py and add another question!

Test it: Run this in Python. Type “4” or something else—what happens?

Chapter 5: Showing Off with GitHub Pages

Turn your repo into a website—like this e-book!

How to Set Up This Page

  1. Create a repo (e.g., “GitHubPythonGuide”).
  2. Add this file as index.html.
  3. Go to “Settings” > “Pages”, set “Source” to “main” branch, and save.
  4. Visit username.github.io/GitHubPythonGuide after a minute!

Chapter 6: Python + GitHub Project – Number Guessing Game

Let’s make a game and share it!

Example 3: Number Guessing Game

1. Create game.py in your repo with:

secret = 6
guess = int(input("Guess a number between 1 and 10: "))
if guess == secret:
    print("You win! Amazing guess!")
else:
    print("Nope! The secret was", secret)

2. Commit it.
3. Ask a friend to change the secret number and test it!

Test it: Run this in Python. Guess a number—what’s the result?

Chapter 7: Exploring More with Python

Python can do cool stuff! Here’s a loop example:

Example 4: Countdown

Add countdown.py to your repo:

for i in range(5, 0, -1):
    print(i)
print("Blast off!")

Test it: Run this—what does it count down to?
